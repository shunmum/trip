# Futari-Tabi（仮）

カップル・夫婦向けの旅行計画Webアプリケーションのプロトタイプ

## アプリの目的

- カップルが旅行計画での「負担の偏り」による喧嘩を防ぐ
- 「誰が調べたか」を可視化し、感謝を生む
- 計画プロセス自体を思い出として残す

## 技術スタック

- **フレームワーク**: React 18 + TypeScript
- **ビルドツール**: Vite
- **スタイリング**: Tailwind CSS
- **アイコン**: Lucide React
- **ルーティング**: React Router Dom
- **データ管理**: モックデータ（将来的にSupabaseを導入予定）

## 機能

### 1. Scrap（スクラップ）
- 気になるURLやメモを保存
- 提案者（UserA/UserB）を明示表示
- カテゴリ別に整理

### 2. Plan（プラン）
- 地図表示エリア（将来的にLeafletで実装予定）
- スポットリスト管理
- 予約ステータス表示（未予約/予約済）

### 3. Guide（旅のしおり）
- **持ち物リスト**: 担当者別チェックリスト
- **スケジュール**: タイムライン形式の予定表
- **基本情報**: ホテル、連絡先、交通手段

### 4. Memory（思い出）
- 写真ギャラリー
- 感想の記録
- 旅行統計

## セットアップ

### 前提条件
- Node.js 18以上

### インストール

\`\`\`bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# ビルド
npm run build

# ビルドしたアプリのプレビュー
npm run preview
\`\`\`

開発サーバーが起動すると、[http://localhost:5173](http://localhost:5173) でアプリにアクセスできます。

## UI/UX の特徴

### モバイルファースト
- スマートフォンでの利用を想定したレスポンシブデザイン
- タッチ操作に最適化されたUI

### ユーザー可視化
- UserAとUserBを色分けで識別（オレンジ系/ブルー系）
- すべてのアクションに担当者アイコンを表示
- 公平性と貢献度の可視化

### 配色
- **Primary（オレンジ系）**: UserA専用カラー
- **Accent（ブルー系）**: UserB専用カラー
- **白ベース**: 清潔感とワクワク感を演出
